# BRUTAL.AGENCY

**// NO FLUFF. NO NONSENSE. JUST HIGH-VOLTAGE GROWTH.**

This is the official repository for BRUTAL.AGENCY, a modern brutalist marketing collective. We build digital weapons for the bold.

## TECH STACK
- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Motion**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## DESIGN PHILOSOPHY
We embrace the **Premium Brutalist** aesthetic:
- High-contrast color palettes (Neon Green, Deep Black, Soft Gray).
- Bold, oversized typography (Inter, Black weight).
- Stark, intentional layouts with rigid borders and "brutal" shadows.
- Purposeful, staggered entry animations.

## GETTING STARTED
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Access at: `http://localhost:3000`
